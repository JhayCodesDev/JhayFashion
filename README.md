# Jhay Fashion 🌟

A responsive fashion website designed to showcase products with a clean, modern interface.

---

## 🔹 About the Project

I designed and built **Jhay Fashion**, a fashion e-commerce webpage. The project includes:  
- Curated product images with names, descriptions, and prices  
- Fully responsive layout for **mobile, tablet, and desktop**  
- User-friendly interface focused on seamless browsing and visual appeal  

This project demonstrates my skills in **HTML5** and **CSS3**, with an emphasis on responsive design.

---

## 💻 Tech Stack

- **HTML5** – Semantic structure and content  
- **CSS3** – Styling, layout, and responsive design
